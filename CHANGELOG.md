# Changelog

## 0.4.0 (2026-05-15)

### Added

- RSS feed generation script using the `feed` package, with support for multiple locales (English and Portuguese).
- Environment variable support for site URL configuration in the RSS feed generation. `SITE_URL` can be set in a `.env` file or will default to the Vercel URL or a placeholder.
